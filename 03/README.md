# SQL Aggregate Functions

This project was created as part of my SQL learning journey to practice **Aggregate Functions**.

In this project, I created a simple **books** database and wrote different SQL queries to analyze the data using aggregate functions.

## Database

The project uses a **books** table with the following columns:

* Book ID
* Title
* Author
* Publisher
* Category
* Language
* Price
* Stock
* Published Year

The database contains **25 sample books** from different categories.

---

## SQL Queries

### COUNT()

* Count all books
* Count books by category
* Count unique categories
* Count unique publishers
* Count books matching specific conditions

### SUM()

* Calculate the total price of all books
* Calculate the total stock
* Calculate the total price by category
* Calculate the total stock by category

### AVG()

* Find the average book price
* Find the average stock
* Find the average price by category

### MIN()

* Find the lowest book price
* Find the oldest publication year
* Find the lowest price in a category

### MAX()

* Find the highest book price
* Find the newest publication year
* Find the highest price in a category

### Bonus

* Use multiple aggregate functions in a single query
