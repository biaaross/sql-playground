This project contains a realistic relational database schema, sample data, and various SQL queries covering beginner and intermediate SQL concepts.

# 🗄️ Database Schema

The database consists of five tables:

- Departments
- Instructors
- Students
- Courses
- Enrollments

Relationships:

- One department has many courses.
- One instructor teaches many courses.
- One student can enroll in many courses.
- One course can have many students.
